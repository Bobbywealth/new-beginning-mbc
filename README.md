# New Beginning Missionary Baptist Church

Static website for New Beginning Missionary Baptist Church.

## Stack

- Single `index.html` (no build step)
- Static asset: `assets/logo.jpg`
- Render: static_site (free tier)

## Local preview

```bash
open index.html
# or
python3 -m http.server 8000
```

## Deploy

Pushing to `main` triggers Render auto-deploy.

## Files

- `index.html` — full page (HTML + CSS + body)
- `assets/logo.jpg` — church logo (extracted from design spec)
- `render.yaml` — Render Static Site config
